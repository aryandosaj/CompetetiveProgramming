# BIT GO Server

BIT GO Server is a server server written in flask.

## Installation

Make sure flask is installed in your system.
```python
    pip install flask
```
## Dependencies

Install these python libraries before running the server

* BeautifulSoup
* urllib.request
* selenium
* PIL
* pytesseract
* parser
* pickle


## Usage

Run webhook.py using the following commands:
```python
python3 webhook.py
```
